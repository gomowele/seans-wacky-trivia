# Sean’s Wacky Trivia 🤪😎🫡

A multiplayer trivia game for coworkers with avatars, live music, timed questions, and leaderboard.

## 🔧 Run Locally

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
pip install fastapi uvicorn python-multipart
uvicorn main:app --host 0.0.0.0 --port 8000
```

## 🚀 Deploy
Frontend → Vercel  
Backend → Render
